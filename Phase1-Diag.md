| Initiation by Farmer           |     | Sample and Consent Management  |

                                               |

                                    | - Auction listing is created  |
                                    +-------------------------------+
                                               |

                                    +-------------------------------+

                                    | - Buyers place bids           |

                                               |

                                    +-------------------------------+
